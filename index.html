Projects

Below is a comprehensive portfolio of the projects I have completed throughout my academic journey.

Movie Details CLI App

This Command Line Interface(CLI) application retrieves movie information from the OMDb API. It asks the user to input a movie title and the returns a JSON object containing details such as the title, release year, IMDb rating, language, and plot.

ApexCare Solutions

ApexCare Solutions specailizes in managing pre-paid electricity meters through tailored service contracts. Using advanced technology, it streamlines job management, real-time tracking. and service monitoring. Its centralized platform integrates automation, analytics, and mobile-friendly tools to enhance technician-client interactions, ensuring efficiency and customer satisfaction.

Loan Application System

This CLI application is designed to manage loan details such as loan number, customer information, loan amount, term, and interest rate. With built-in validation and calculation methods, it ensures proper loan data handling and provides an easy-to-read output of loan information. Its flexible structure supports various loan types, centralizing essential loan functionality for efficient and streamlined loan management in any loan application system.

Data Capturing System

The Data Capturing System is a project designed to simplify the process of collecting and managing student data related to their enrolled modules. The platform allows users to input, view, and update student information, including personal details and their associated modules. It also enables the tracking of module-specific informaiton. The system helps educators and administrators efficiently manage student records, ensuring accurate and up-to-date data for better decision-making and academic planning.

Validation Form

This form system efficiently collects and validates user information, including personal details and location. Using built-in validation for names, email address, and dropdown selections, it ensures accuracy and compliance, with set criteria. The form provides a dynamic user experience by displaying real-time validation feedback and summarizing submitted data. Its intuitive interface improves user interaction while enabling easy from submisiion and reset, ensuring seamless data entry and presentation.

Issue Tracking System

The Issue Tracking System is a project designed to streamline the management and tracking of issues, tasks and bugs within a project. This platform enables users to efficiently report new issues, assign them to team members, set priorities, and monitor their resolution in real-time. Key features include the ability to add, view, and update issues, as well as categorize them by status, priority, and deadlines. This system ensures effective issue management and fosters collaboration among teams, making it an essentail tool for maintaing smooth project workflows.

Colour Sorting Robotic Arm with Detection System

For my Final Year group project, we developed a robotic arm sorting system powered by an object detection algorithm. The system utilised computer vision technologies like YOLO(You Only Look Once) and OpenCV to identify objects based on attributes such as colour, shape, and size. A Raspberry Pi Pico controlled the robotic arm, allowing it to perform precise movements to pick, sort, and place items into designated categories. Unfortunately, due to version control issues and unauthorised tampering with teh robotic arm, there is no GitHub Repository available for this project.
